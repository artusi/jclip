What is JClip?

It is a simple javascript function to simulate a basic Flash MovieClip in Html.

Example:

  • 1 step: Make your animation on flash;

  • 2 step: Generate a SpriteSheet (i am using SpriteSheetCreator);

  • 3 step: Use Jclip to simulate;

Documentation:

Property

  • element : String

    Div id

  • url : String

    Image url

  • frameWidth : Number

    Spritesheet frame width (swf width)

  • options { frameRate: Number = 40 }

    Extra options, frameRate set jclip framerate
